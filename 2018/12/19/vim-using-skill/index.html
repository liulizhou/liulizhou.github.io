命令

  1. Home键或数字0快速定位到行首，End键定位到行末

  2. $键前加数字表示移动的行数，1$表示当前行的行尾，2$表示当前行的下一行的行尾

  3. gg跳转到第一行，Shift+g, G跳转到最后一行

  4. Ctrl + s 锁定当前终端， Ctrl + q 解锁
